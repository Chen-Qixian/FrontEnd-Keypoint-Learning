# 先先的音乐播放器

### 使用方法

在浏览器打开demo.html

### 说明

采用纯原生JS编写了一个带有日食效果的可调节音量的音乐播放器

### 练习要点

* dom对象操作
* 鼠标事件绑定
  * onmousedown
  * onmouseup
  * onmousemove
* 面向对象编程
* H5标签<audio>的使用
  * audio.volume
  * audio.play()
  * audio.pause()
* 拖拽效果练习
  * 原对象e获取鼠标位置
* 一些CSS样式的练习
  * box-shadow
  * hsl颜色函数
  * 自适应圆形
